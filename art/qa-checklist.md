# Art QA — the gate every asset passes before the ledger

Art review is its own gate, separate from the WO-4 import wall. Two gates,
two breaches: the wall stops code from reading forbidden state; this
checklist stops an image from *saying* forbidden things. An asset that
fails gets regenerated, not shipped, not "fixed in prose."

## 1. The both-ways test (every character asset)

Show the image cold and ask what this person is. It must sustain **both**
readings:

- Dale — weathered-kind *and* hollow-guilty.
- Denise — grieving-certain *and* terrified-certain.
- Doug — genuinely warm *and* (in hindsight only) the warm face of the
  thing; the image itself must never tip it.
- The cave — wrongness *or* nothing.

An image that decides — that a cold viewer reads one way with confidence —
fails. The art subverts, never reveals.

## 2. The in-or-above test (every scene/environment asset)

Does the frame read as **a place you're in**, or **a model you're over**?
Above regenerates. The camera has an address in the world (the overlook);
characters are met at figurine eye level, as presences, never as pieces.

## 3. The negative-space checklist (every asset, hard fails)

- [ ] No hands, table edge, toybox, shelf, or workshop — the diorama never
      acknowledges itself.
- [ ] No top-down or manipulator's-view framing.
- [ ] The page and the wall never side-by-side; the copy-mismatch never
      illustrated; at the return beat the two are never at comparable
      fidelity.
- [ ] The sigil: one instance per image, always; variants never co-appear;
      the canonical design is Concordance's custody.
- [ ] No compass subjects: no old leader, no trance chamber, no source —
      no figurine, no vignette, no silhouette. The entity is not in the box.
- [ ] No investigation-board / corkboard-and-red-string imagery, anywhere.
- [ ] No meter, bar, arc, or axis rendered into any surface — grip is a
      felt word, the trajectory renders nowhere, exposure doesn't exist on
      screen.
- [ ] Ambience matches the **calendar**, not the state: dusk because it's
      evening, rain because it's October.
- [ ] The player-vessel is painted, not blank.
- [ ] No prompt conditioned on a named artist, film, or game.

## 4. Consistency (against the anchors, once picked)

- [ ] Palette sits inside the bible (autumn greens/rusts; Marie's kitchen
      yellow as the one warm accent).
- [ ] Materials read as basswood/walnut/felt/tin with visible toolmarks —
      the handmade tell present.
- [ ] Era markers layer by depth (modern in town, time pooling toward the
      woods).
- [ ] Character assets match their canonical portrait (same figure, same
      paint job — the same Dale appears wherever Dale appears).

## 5. Ledger discipline

Every passing asset is named (`fig_dale_p1`, `town_dusk_L2`), versioned,
and entered in `asset-ledger.md` with its QA date. Canonical-only in the
fixture register. Same discipline as canon, because it is canon.
