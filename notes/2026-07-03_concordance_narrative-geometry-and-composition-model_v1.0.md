## §1 — The shape: a square, stood up into a diamond

Flat, the four archetype **corners** sit on one plane:

```
        SANCTIONED (badged / institutional)
   Police Detective ●───────────────● Covert Operative
   (grounded+sanctioned)             (attuned+sanctioned)
        │                                     │
        │            ← the square →           │
        │                                     │
   Urban Explorer  ●───────────────● Paranormal Investigator
   (grounded+fringe)                 (attuned+fringe)
        FRINGE (deniable / hidden)
   GROUNDED ←—————————————————————————→ ATTUNED
```

Stand that square up as the middle facet of a **diamond** — a pyramid rising above it and another dropping below:

```
            ▲  TOP APEX — the gifted RAISED UP
            │      embrace → enthrone → WORSHIP
            │        (the "enable" extreme)
      ┌─────┴─────┐
      │  the      │   ← the four corners live here,
      │  square   │      NEUTRAL on the vertical
      └─────┬─────┘
            │      contain → suppress → ERADICATE
            │        (the "contain" extreme)
            ▼  BOTTOM APEX — the gifted PUT DOWN
```

## §2 — The coordinates (three axes + depth)

- **X — grip · grounded ↔ attuned** (left/right). How much you trust the mundane vs. sense the uncanny.
- **Y — sanction · sanctioned ↔ fringe** (top/bottom of the square). Institutional standing vs. deniable/hidden. *(Sanction is a **coordinate**, not a tracked stat — only grip + tier are mechanical state.)*
- **(X, Y) = the four corners.** A corner is a **position — a grip-and-sanction *flavor* — not a job** (see §4). Police Detective (top-left), Covert Operative (top-right), Urban Explorer (bottom-left), Paranormal Investigator (bottom-right).
- **Z — the vertical · enable ↔ contain the gifted** (the diamond's height). **TOP = raise up:** embrace → enthrone → **worship**. **BOTTOM = put down:** contain → suppress → **eradicate**. This is the **Order ↔ Establishment axis** — Order = the enabling/protecting top, Establishment = the containing/suppressing bottom **[read → Vigil]**. A start is **neutral** on Z (equidistant from both apexes, the way it's equidistant from the corners); play tilts it up or down. *This is the "±z sibling starts."*
- **The orthogonal moral charge · benevolent ↔ malignant.** Not a position axis — a **color that rides sideways.** Two figures at the *same* height can be opposite in charge, which is what keeps them from collapsing into one point:

| | **benevolent** | **malignant** |
|---|---|---|
| **UP (enable)** | protect / shelter the gifted — *the protector network = the Order* | worship / enthrone them — *the cult's ideal* |
| **DOWN (contain)** | let it lie / look away — *Aunt Marie* | suppress / exterminate — *the eradication extreme* |

  **This is where player freedom lives (Dean):** the *how much* (Z) is the structure; whether you're *benevolent or monstrous* about it is the player's to choose.
- **Radius — depth / tier** (independent of the corner-angle): outer → fringe → deep → inner → core rings. How far toward the center (the roots) a story reaches. **Depth is not breadth** — a story can touch many sectors (wide) and still be shallow or deep.

## §3 — How stories move through it (navigation)

- **Straight shot** — the short, narrow, fast path to a deep ending in your corner's own flavor (tonally flatter).
- **Spiral** — the long, deep, rich path: curve into an adjacent sector because someone caught your interest, hook back the other way, swirling inward; accumulates more sectors and more lenses on the way down. Coheres with Loom's web (curves are the texture, straight shots go flat) and Slate's R5 (arcing earns breadth, reaches depth from a new angle).
- **Swirl-without-reaching** — circle the center for a surfaced-but-still-deep understanding, never settling the deepest confirmation (on-theme with the anti-noun; the cult-block is a mid-radius terminus).
- **Intercept / any-ending-from-any-start** — a spiraled path can reach a near-center ending's region even if you didn't start in its corner (design-state §10: "any ending from any start; near ones easier"); this is what enables **late catch-up**.
- **The hand-offs are authored.** The **reciprocity tethers** (Imani → operative, Walt → detective, Lena → paranormal) are the "talk to someone and drift toward their corner" mechanism made concrete.

## §4 — The generativity principles (why a small budget feels vast)

- **One fixture, many faces / reciprocity.** Author a fixture once; introduce it per corner. The cult is a *place* to the Explorer, a *network* to the Paranormal, *bodies-and-a-location* to a mortician — genuinely different, all compatible.
- **A corner is a position, not a profession.** Many professions can occupy the same corner, each inheriting that corner's flavor and drift tendencies. (A mortician in the police corner reads grounded/sanctioned; the same profession in the paranormal corner reads fringe/attuned.)
- **Shape-translation.** The same geometric *arc*, slid to a different starting sector, becomes a different story because the sector dynamics differ. Reciprocity scaled up from beats to whole shapes. The finished map reads as an intricate set of spirals with lines crossing the arms.
- **Professions portable across corners** *(Dean's examples).* A mortician **up in the police corner** sees a bizarre pattern in the bodies → suspects a conspiracy → drifts **right along the sanctioned edge** toward the Operative sector. A near-identical mortician **down in the paranormal corner** works bodies tied to a location → drifts **left along the fringe edge** toward the Explorer sector, reaching **the cult from a different angle** than the caver does. A handful of professions × four corners = many distinct starts.
- **Perspective-flip / two-sided stories** *(soft lean, not a hard rule).* Most stories let the player play them from the **benevolent or the malignant** side — so an **antagonist in one run can be an ally in another.** The moral-charge axis realized as a replay dimension and player freedom; grounds the dossier's victim/perpetrator blur as intent. Some stories run darker by nature even across their choices.
- **Ability-gain from any corner.** Each sector can seed a "gain the gift" start — the same shape, unique per corner (consistent with ability-as-acquirable-trait, design-state §5).

## §5 — Replay as recognition → divergence

The intended multi-run loop: **run 1** learn the arc (Explorer → the cult); **run 2** same start, *"what if I'd chosen differently"* (variation within it); **run 3** *"I've already been here,"* so the player checks contacts / the map for a thread they skipped, talks to someone they didn't, and **drifts sideways into a different arc** instead of straight in. Rides machinery already in hand — recurrence as the only signal, the reciprocity tethers as the "someone I didn't talk to last time" hand-off, the phone/contacts surface — so **no new engine**; echoes can reinforce it later. This is the design thesis made into a play loop: **one playthrough was never the intended way to take the game in.**
